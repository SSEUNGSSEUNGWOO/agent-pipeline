# 캘리브레이션 예제 파일 형식

## 파일 구조

```
examples/
├── sample1.txt          # 지원서 원문 (UTF-8)
├── sample1_scores.json  # 사람 평가 점수
├── sample2.txt
├── sample2_scores.json
└── ...
```

## 점수 JSON 스키마

```json
{
  "evaluator": "평가자 이름 (선택)",
  "scores": {
    "지원동기": 8,
    "AI·데이터 활용 경력": 15,
    "현안 연결성": 3
  },
  "notes": "평가자 메모 (선택)"
}
```

## 주의사항

- `scores` 키의 항목명은 루브릭의 `items[].name`과 정확히 일치해야 합니다.
- 불일치 시 유사도 0.6 이상인 항목에 자동 매핑됩니다.
- 유사도 미달 항목은 스킵됩니다.
- 최소 3개 이상의 캘리브레이션 샘플을 권장합니다.
